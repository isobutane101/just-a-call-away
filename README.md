# Just a Call Away

An emergency preparedness initiative helping elderly South Asian immigrant communities understand when and how to call 911.

**Website**: [justacallaway.org](https://justacallaway.org)

## Mission

This project addresses a critical gap where language barriers, fear of authorities, and unfamiliarity with emergency systems prevent immigrant families from seeking timely help in life-threatening situations.

## Features

- **Multilingual Resources**: Emergency guides in English, Hindi, and Gujarati
- **Downloadable Flyers**: PDF guides covering when to call 911, critical symptoms, and communication tips
- **Key Information**: 140+ language interpreters available, immigration status not asked

## Technologies

- HTML5, CSS3 (no frameworks)
- Google Fonts (Playfair Display, Source Sans 3)
- GitHub Pages hosting
